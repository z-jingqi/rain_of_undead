# 玩家系统

## 基础属性
- 移动速度：中等
- 初始生命值：3点
- 射击方式：自动
- 子弹伤害：基础1点

## 角色设计

### 1. 突击手（Assault）
- 基础属性：
  * 移动速度：快
  * 生命值：2点
  * 子弹伤害：1点
- 主动技能：火力全开
  * 效果：短时间内射击频率翻倍
  * 冷却时间：30秒
  * 持续时间：5秒
- 被动技能：快速装填
  * 效果：射击频率永久提升20%
- 升级树：
